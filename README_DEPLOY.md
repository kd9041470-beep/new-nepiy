# نشر المشروع على GitHub Pages (Vite + React)

تم تجهيز المشروع للعمل على GitHub Pages بدون أخطاء شاشة بيضاء من خلال:
- ضبط `base: './'` في إعدادات Vite لروابط الأصول النسبية.
- استخدام `HashRouter` بدل `BrowserRouter` لتفادي 404 عند التحديث.
- إنشاء `404.html` تلقائياً بعد البناء.
- إعداد GitHub Actions للنشر التلقائي.

## خطوات سريعة
1. ثبّت الاعتمادات:
   ```bash
   npm ci
   ```
2. ابنِ المشروع:
   ```bash
   npm run build
   ```
3. فعّل GitHub Pages عبر الإعدادات واختر "GitHub Actions".
4. ادفع إلى الفرع `main` وسيتم النشر تلقائياً.

> ملاحظة: إن رغبت لاحقًا بالعودة إلى `BrowserRouter`، أضف `basename` الصحيح واحتفظ بملف `404.html`.
